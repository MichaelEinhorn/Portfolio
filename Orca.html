Orca IoT AWS Developer Intern

As a rising Junior I was an intern at Orca IoT, a startup developing software to manage construction sites more efficiently. A use case for computer vision is identifying images where heavy machinery is in use, such as a cherry picker, to monitor for OSHA compliance. I prototyped an object detection system both on the edge with a Jetson Nano, and in the cloud with AWS Inferentia. I solved the problem of processing a 4k image with a model that uses a much smaller image size, by slicing the image into overlapping tiles and merging the model's detections from each tile into a single set for the larger image. This created a pipeline where a different image could each be at each step at the same time to maximize throughput which minimizes cost. This system was both more accurate and less expensive than using AWS Rekognition when tested on our in-house dataset. See a marketing demo of this solution below. I made the animation and the code for using the model, I am not the presenter.
