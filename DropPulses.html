Dropped Pulses Signal Analysis Algorithm

I developed an algorithm to detect dropped pulses in sequences of Pulse Repetition Intervals (PRI), based on an algorithm used for Spellcheck.
A PRI is the interval between adjacent pulses in a sequence. When a pulse is dropped the resulting new PRI in the sum of the two original PRIs.
While checking this sum is simple, finding where to apply it within a larger sequence and then finding the best match sequence is time consuming to do manually. An example is comparing a PRI sequence of 4, 2, 3, 5 and 3, 1, 2, 3, 5. The 4 at the beginning represents a dropped pulse as its the sum of the PRIs of 3 and 1. Simply checking the PRIs one by one results in every PRI being wrong, despite there only being one dropped pulse.
The issue is that a dropped pulse results in a shift of the whole sequence and the algorithm needs to take this into account.

I have seen similar problems in Spellcheck where a deleted or inserted letter shifts the rest of the word and in Biology with identifying frameshift mutations. While both the domain and use case are different from signal analysis, the shared shape of the problem is comparing sequences while considering shifts.
The Wagner Fischer algorithm is able to identify the edit distance with operations of insertion deletion and substitution, and was used as an early spellcheck algorithm. It is a Dynamic Programming algorithm that fills out a 2D table to find the optimal alignment for edit distance. The Wagner Fischer algorithm builds this alignment incrementally and selects the shortest path to each cell from the previous cells.

Dropped Pulse Algorithm


Green - Start  Blue - Substitution
Red - Dropped Pulse  Purple - Extra Pulse

A substitution compares one observed letter with one reference letter while a deletion compares zero observed letters with one reference letter. The number of rows and columns an arrow crosses is how many elements that operation compares.
While keeping the same general structure of the algorithm, it can be modified to work with PRI sequences. The observed PRI sequence is allowed to start anywhere in the reference and wrap around the end, so the entire top row is a starting position. To work with numerical values, substitution uses a score of squared error instead of a constant value. Dropped Pulses match the sum of two reference PRIs with one observed PRI, so this corresponds to the red arrow crossing 2 columns and 1. Extra pulses work similarly but compare two observed PRIs with one reference PRI. The best distance on the bottom row corresponds to the best alignment, which in this case contains one dropped pulse.
